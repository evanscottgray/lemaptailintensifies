# how2maptail

Be sure to install node.js first!


```shell
git clone https://github.com/evanscottgray/lemaptailintensifies
cd lemaptailintensifies
npm install
tail -f /path/to/aggregated/logs/of/all/the/things | maptail -p $port -h $interfaceIp
```
